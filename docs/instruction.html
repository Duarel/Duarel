Как зайти на сервер Duarel

Вы когда-нибудь задумывались, как правильно подключиться к серверу Minecraft? Если да — тогда давайте разберёмся шаг за шагом.
Всё максимально просто: Наш сервер работает на версии Minecraft Java Edition 1.20.2. Подключиться можно как с лицензионной версии игры, так и с пиратской.

Шаг 1. Скачайте лаунчер Minecraft

Для начала вам понадобится лаунчер:

Лицензионная версия
|
| —  Если у вас есть официальная версия игры, скачайте лаунчер с официального сайта Minecraft и войдите в свой аккаунт.

Пиратская версия
|
| —  Если у вас нет лицензии, можно использовать сторонний лаунчер.
| —  Один из самых популярных — TLauncher.
|
| —  Введите в Google запрос "TLauncher".  Перейдите на официальный сайт.  Скачайте установщик и установите лаунчер.
| —  Запустите программу и выберите версию 1.20.2 (обязательно Java Edition).

Шаг 2. Выберите правильную версию игры

В лаунчере выберите версию 1.20.2. Убедитесь, что это именно Java Edition
Нажмите кнопку "Играть". Дождитесь полной загрузки игры.

Шаг 3. Добавьте сервер

После запуска игры нажмите "Сетевая игра".
Нажмите "Добавить сервер" и в поле "Имя сервера" введите любое название (например: Duarel KitPvP).
В поле "Адрес сервера" введите Duarel.kitpvp.su
Нажмите "Готово".
Выберите сервер в списке и нажмите "Подключиться".
Готово! Вы на сервере.

Допустимые моды

Вы можете использовать моды, которые не изменяют геймплей и не дают преимущества над другими игроками.
Разрешённые примеры:

OptiFine — улучшает FPS и добавляет настройки графики
Just Enough Items — удобный просмотр рецептов
TL Skin — скины для пиратской версии
Simple Voice Chat — голосовой чат
Запрещены любые читы, x-ray, fly-моды, автокликеры и другие модификации, дающие преимущество


Если не получается зайти

Проверьте:
  • Версия игры точно 1.20.2

  • Включён ли интернет

  • Нет ли лишних модов

  • Правильно ли введён IP

Если сервер не отображается, попробуйте перезапустить игру или лаунчер.

Ждём вас на сервере!
